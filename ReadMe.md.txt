Backend
Step 1: Download and extract the zip flie
step 2: Open the "web api" folder in visual studio
step 3: Go to appsetting..json and change the sever name to your local server name
Step 4: Run migrations to create the tables on your database
Step 5: Make sure the urls are configured
Step 6: Run the server on https

Frontend
Step 1: Download and extract the zip file 
Step 2: Open "Reimbursement_frontend" folder on visual studio
Step 3: Run "npm i'/'npm install" to install all packages
Step 4: MAke sure the service urls are configured to your server port in the services/ folder
Step 5: Run 'ng serve --o' for a dev.
